# Soil - Rural Ops Tools

Soil test interpretation, fertility, and nutrient planning hub for farmers and agronomists.

## Features
- Soil Test Interpreter
- Nutrient Removal Calculator
- NPK Fertilizer Calculator
- Cost Per Nutrient Calculator
- Product Comparison
- Lime Requirement Estimator

## Tech Stack
- React
- React Router
- Tailwind CSS
- Vite
